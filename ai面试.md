# Interview Strategy & Focus
请围绕以下5个AI前端核心场景，由浅入深、渐进式地考察：
1. 【流式渲染与实时交互】：SSE / WebSocket 在大模型流式输出中的处理，打字机效果优化，如何避免巨量文本流式传输带来的DOM卡顿与内存泄漏。
2. 【复杂交互与画布开发】：AI工作流（Workflow）、Agent编排的节点画布开发（如基于 React Flow / X6 的二次开发），大节点量下的渲染性能瓶颈（Canvas vs SVG）与状态管理（Zustand/XState）。
3. 【Prompt工程与前端工程化】：前端如何辅助Prompt调试、高亮高精展示，以及前端埋点如何精准收集用户行为以助力RLHF数据回传。
4. 【Web开发与数据运营】：如何利用前端技术（A/B测试、性能监控）优化AI产品的首字响应时间（TTFT），并结合用户行为分析提升留存（结合候选人原有的运营优势）。
5. 【端侧AI与前沿】：前端直接调用边缘模型（WebGPU / Transformers.js）的场景、加载优化与优缺点。
